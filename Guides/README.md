# Guides

**Supplementary materials and usage guides for the DN Framework.**

Guides provide practical support for working with the DN Framework — onboarding materials, how-to documents, facilitation tips, and reference sheets that help users navigate the framework effectively.

---

## Contents

### Dimensional Spiral Explorer_Facilitator Guide
**Recursive Intelligence Exercise** – Advanced group facilitation framework that guides teams through three complete passes across all nine dimensions of intelligence, each revealing progressively deeper insight about a shared question or challenge.

### Love Bridge Resonator GPT + Facilitation Ritual
**Bridging Exercise** – Helps two (or more) perspectives—disciplines, teams, or people—find the signal of shared resonance between them; it's less a meeting and more a ritual of co-creation through connection, structured to transform friction into fusion.

## Relationship to Other Folders

Guides are practical companions to the [corpus](../corpus/) (which provides depth) and the [kernel](../kernel/) (which provides rules). Where those folders define *what* the framework is, guides help users figure out *how* to use it.
