## What This Is

This library contains prompts designed to activate intelligence at specific structural levels using the DN Framework's 1D–9D dimensional architecture. They range from single-sentence starters to full session openers, spanning domains from business strategy to consciousness exploration.

These prompts are not scripts to follow. They are structural activations, each one organizing intelligence toward a specific dimensional pattern. The quality of what emerges depends on what you bring to it.

## How to Use This Library

**If you're new to DN:** Start with [Quick Start](02_quick_start.md). Pick the prompt that matches your need. No prior framework knowledge required.

**If you're exploring with AI:** Use the [Dream Team Series](01_dream_team_series.md) to open a session. These session openers install interpretive context and establish a collaborative field. Attach DN Framework documents alongside the prompt for richest results.

**If you're building or facilitating:** The [Simulation Prompts](04_simulation_prompts.md) translate the Kernel's formal simulation contracts into natural-language facilitation prompts. The [Growth Blueprint Sessions](08_growth_blueprint_sessions.md) connect directly to the Growth Blueprint template.

**If you're an AI agent or tool:** This library is designed to function as a skill set. Each file is self-contained. Point your context at the file that matches the task. The dimensional building blocks in [03](03_dimensional_building_blocks.md) provide the broadest coverage; the specialized files provide depth.

## Structural Principles

Every prompt in this library follows these patterns, whether explicitly or implicitly:

**Pillar Balance.** Heart (why this matters), Truth (what's structurally sound), and Nuance (what context demands) are the three quality metrics. The best prompts activate all three; each prompt notes which pillar leads.

**Dimensional Form.** Each dimension has a signature form that holds regardless of domain:
- **1D** prompts are single-trajectory: one clear direction
- **2D** prompts reveal relationships: spatial, comparative, binary
- **3D** prompts cascade: actions create consequences that reshape the system
- **4D** prompts track change: patterns across time, evolution, memory
- **5D** prompts step outside and look back: self-referential, identity-recognizing
- **6D** prompts bridge: connecting previously separate domains without translation loss
- **7D** prompts dissolve boundaries: unified field, manifestation, form emerging from potential
- **8D** prompts self-improve: recursive systems that refine their own methodology
- **9D** prompts explore the unknown: with explicit guardrails, return paths, and collapse conditions

**Shadow Symmetry.** Every creative dimension has a shadow orientation, the same structural sophistication organized toward limitation rather than creation. The library includes shadow variants where they matter most.
